# Die Walzenbrüder — Website (statische Vorlage)

Beschreibung
- Statische, responsive Website für den Jägerzug "Die Walzenbrüder" (BSV Grevenbroich).
- Modernes Design mit Schützen‑Charakter (Farben: Grün / Weiß / Gold).

Dateien
- index.html — Startseite (Hero, Kurzvorstellung)
- about.html — Über uns / Geschichte
- members.html — Mitglieder / Team
- events.html — Termine / Events (Platzhalter)
- contact.html — Kontakt, Impressum & Datenschutz (Platzhalter)
- css/styles.css — eigene CSS‑Anpassungen
- assets/logo.svg — SVG‑Logo (basierend auf deinen Bildern)

Deployment (lokal testen)
1. Alle Dateien in einen Ordner legen.
2. index.html im Browser öffnen.

Deployment zu GitHub Pages
1. Neues Repository erstellen (oder vorhandenes verwenden).
2. Dateien pushen (Branch `main`).
3. In den Repo‑Einstellungen GitHub Pages aktivieren (Quelle: `main` / `/ (root)`).
4. Seite ist erreichbar unter `https://<user>.github.io/<repo>/` (ein paar Minuten warten).

Hinweis zu Inhalten & Rechtlichem
- Impressum/Datenschutzhinweise sind Platzhalter — bitte mit korrekten Kontaktangaben ersetzen.
- Logo wurde als SVG basierend auf deiner Vorlage erstellt. Wenn du das originale Logo (als Foto oder höherauflösendes SVG) verwenden willst, lade es bitte hoch oder gib mir Bescheid.
